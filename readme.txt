Light annotation-boxes for annotations, information or small notes.

== Description ==
Create light annotation-boxes for annotations, information or small notes.
[info]...[/info]
[pin]...[/pin]
[note]...[/note]
[quote]...[/quote]

Add `showicon="true"` as Parameter to add a dashicon to the annotation.
Example: [info showicon="true"]Important information[/info]

== Installation ==
1. Upload folder `mcjh-anotation` to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Ready. Now start adding info-boxes, pin-boxes and note-boxes!
